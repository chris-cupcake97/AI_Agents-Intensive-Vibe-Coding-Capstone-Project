# Security

FundMyDegree is a fixture/offline Kaggle capstone prototype.

For the full security model, see [docs/security.md](docs/security.md).

## Project Safety Boundaries

- Do not commit API keys, passwords, service-account files, or real secrets.
- Do not upload passports, transcripts, bank statements, or other sensitive
  student documents.
- The current app does not send emails automatically.
- The current app does not submit scholarship or university applications.
- The current app does not scrape live websites or call external scholarship
  APIs.

## Reporting Issues

For a public demo repo, report issues through GitHub Issues. Do not include
secrets or private student information in issue descriptions, screenshots, or
logs.
